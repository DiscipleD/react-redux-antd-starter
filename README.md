# react-redux-antd-starter
**React 全家桶 + Ant.Design 脚手架**

## React 全家桶

* react: ^15.3.1
* react-redux: ^4.4.5
* react-router: ^2.7.0
* react-router-redux: ^4.0.5
* redux: ^3.6.0

## Ant Design

* antd: ^1.11.0

## start

```bash
npm install
```

## run dev server

```bash
npm start
```

## build package

```bash
npm run build
```
