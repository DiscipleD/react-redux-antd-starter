# nebula-portal

## env 
node 6.0

## start

```bash
npm install
```

## run dev server

```bash
npm start
```

## run UT

```bash
npm test
```

## See UT coverage

```bash
npm run test:coverage
```

## build package

```bash
npm run build
```
